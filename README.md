# OToggl

Version 0.2.1

[![Actions Status](https://github.com/christophe-riolo/otoggl/workflows/CI/badge.svg)](https://github.com/christophe-riolo/otoggl/actions)

Bindings for Toggl API in OCaml

## Installation

### Using Opam

```bash
opam install otoggl
```

### Using Esy

```bash
esy add @opam/otoggl
```
